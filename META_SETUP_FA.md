# اتصال رسمی Instagram برای کاربران

برای اینکه دیگران هم از سیستم استفاده کنند، نباید همه از یک توکن مشترک استفاده کنند. هر کاربر باید حساب خودش را وصل کند.

## هر کاربر چه چیزی لازم دارد؟

1. اکانت Instagram باید Professional باشد: Business یا Creator.
2. اکانت Instagram باید به یک Facebook Page وصل باشد.
3. کاربر باید از مسیر Meta Developer / Graph API مقدارهای زیر را بگیرد:
   - `IG User ID`
   - `Access Token`

## مسیر امن فعلی

در نسخه فعلی، کاربر داخل پنل ثبت‌نام می‌کند و مقدارهای `IG User ID` و `Access Token` خودش را وارد می‌کند. Access Token در دیتابیس به‌صورت رمزگذاری‌شده ذخیره می‌شود و هر کاربر فقط پست‌های خودش را می‌بیند.

## مرحله حرفه‌ای بعدی

برای محصول عمومی‌تر و راحت‌تر، مرحله بعد باید OAuth رسمی Meta اضافه شود. در آن حالت کاربر به‌جای کپی‌کردن توکن، روی «اتصال به اینستاگرام» می‌زند، وارد Meta می‌شود و مجوز را تأیید می‌کند.

## نکته رسانه

Instagram API فایل را مستقیم از کامپیوتر نمی‌گیرد. عکس یا ویدئو باید URL عمومی داشته باشد، مثل:

```text
https://your-domain.com/photo.jpg
```

اگر بخواهی، مرحله بعدی را با Cloudflare R2 می‌سازیم تا کاربر فایل را داخل همین پنل آپلود کند و سیستم خودش لینک قابل انتشار بسازد.

## چیزهایی که عمداً انجام نمی‌دهیم

- گرفتن پسورد اینستاگرام
- ورود خودکار با مرورگر
- اسکریپت کلیک‌زن شبیه انسان

این روش‌ها ناامن‌اند و ممکن است باعث محدودیت یا بسته‌شدن اکانت شوند.
