# اتصال رسمی اینستاگرام

برای اینکه سیستم بتواند خودش پست بگذارد، باید از مسیر رسمی Meta استفاده شود.

## چیزهایی که لازم داری

1. اکانت Instagram باید Professional باشد: Business یا Creator.
2. اکانت Instagram باید به یک Facebook Page وصل باشد.
3. در Meta for Developers باید یک App بسازی.
4. برای App باید دسترسی‌های Instagram Graph API گرفته شود.
5. در نهایت این دو مقدار را لازم داریم:
   - `IG_USER_ID`
   - `IG_ACCESS_TOKEN`

## نکته مهم رسانه

Instagram API فایل را مستقیم از کامپیوتر نمی‌گیرد. باید عکس یا ویدئو یک لینک عمومی داشته باشد، مثل:

```text
https://your-domain.com/photo.jpg
```

اگر خواستی، مرحله بعدی را طوری می‌سازم که از داخل همین پنل فایل را آپلود کنی و خودش با Cloudflare R2 لینک بسازد.

## چیزهایی که عمداً انجام ندادم

- گرفتن پسورد اینستاگرام
- ورود خودکار با مرورگر
- اسکریپت شبیه‌ساز آدم برای کلیک‌کردن در Instagram

این روش‌ها ممکن است باعث محدودیت یا بسته‌شدن اکانت شوند. این پروژه برای مسیر رسمی و امن ساخته شده است.
